Information:

In order to use the speech sitenza system you need to go to the tab "Speech synthesis" , where the following functions will be available:

- text input, or copying through the buffer in a special field;

- playback of the generated speech signal for the entered text;

- adjusting (selecting) the speaker's voice, tempo, volume.

In order to use the speech analysis system you need to go to the tab "Speech analysis" , where you can get information about the following fine arts subjects (you must click on the "Analyze" button beforehand, and then you can talk):

- command "text" - to get text about the piece of art;

- command "voice" - to listen about the piece of art;

- command "find" - to find webpage.
